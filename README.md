# LKK Interview Question Bank

A trilingual (简体中文 / 繁體中文 / English) interview assessment tool for Lee Kum Kee's V&L competency framework.

## Features

- **Trilingual Support**: Switch between Simplified Chinese, Traditional Chinese (HK), and English
- **Three Position Levels**: IC (Individual Contributor), TL (Team Leader), SL (Senior Leader)
- **144 Interview Questions** across Values, Autopilot Leadership, Leading Change, and Motivation dimensions
- **STAR Framework**: Structured follow-up questions for behavioral interviews
- **Dark Mode**: Eye-friendly dark theme support
- **Export Options**: Export to Word (.doc) or PDF (via print)
- **PWA Ready**: Install as a standalone app on mobile/desktop
- **Responsive Design**: Works on desktop, tablet, and mobile devices

## Usage

1. Select a position level (IC / TL / SL)
2. Choose assessment elements to evaluate
3. View and use interview questions with STAR follow-ups
4. Copy individual questions or export all selected questions
